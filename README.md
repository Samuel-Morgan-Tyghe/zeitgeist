# 🧭 zeitgeist

A twice-daily distilled read on **what's real, what's overhyped, and what's next** in AI engineering — auto-generated from a private "second brain."

**Live reading page:** https://samuel-morgan-tyghe.github.io/zeitgeist/

- `index.md` — the landing page (hype radar + latest TL;DR)
- `digest/` — dated digests + the living `zeitgeist-tracker.md` (hype-lifecycle map)

This is distilled commentary on public AI-engineering news and research; the underlying knowledge base stays private.

> _This folder is the bootstrap copy. To go live: create a public repo named `zeitgeist`, drop these files in at the root, and enable Pages (Settings → Pages → `main` / root)._
